这是从 `E:\TJUTCM\Activities\中国大学生计算机设计大赛\老年友好端\老年友好端` 的 Flutter 原型迁移出的网页子站，作为现有静态站点的用户端模块。

- 桌面预览：`elder/web-preview.html`
